# Downloads werden nicht hier gehostet

Binaries liegen auf GitHub Releases und werden direkt von dort geladen:
https://github.com/ninoborn1-stack/DocSorterV2/releases/latest

Diese Umstellung verhindert Bandwidth-Kosten auf dem Landing-Host.
Die Landing-Page lädt `version.json` (klein) und zeigt die aktuellen
GitHub-Release-Download-URLs an.
